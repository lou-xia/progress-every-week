弄清楚了用户态地址空间结构：

| VA 区间                                             | 作用                          | 用户态访问状态     |
| ------------------------------------------------- | --------------------------- | ----------- |
| `0x0000_0000_0000_1000` ~ `0x0000_0000_000c_7000` | 用户程序 `.text`                | 可读 / 可执行    |
| `0x0000_0000_000c_7000` ~ `0x0000_0000_000c_c000` | 用户程序 `.data/.bss`           | 可读 / 可写     |
| `0x0000_0000_0400_0000` ~ `0x0000_0000_0409_4000` | 用户共享库 / 动态链接器               | 可读 / 可执行    |
| `0x0000_0000_0409_4000` ~ `0x0000_0000_0409_8000` | 共享库数据区                      | 可读 / 可写     |
| `0x0000_0000_4000_0000` ~ `0x0000_0000_4001_0000` | vvar / 共享时间页（准-vDSO）        | 可读 / 可写     |
| `0x0000_0000_6000_1000` ~ `0x0000_0000_6000_2000` | 用户 trampoline / syscall 跳板  | 可读 / 可执行    |
| `0x0000_0003_fff8_0000` ~ `0x0000_0004_0000_0000` | 用户栈                         | 可读 / 可写     |
| `0xffffffc0_0000_0000` ~ `0xffffffc0_000f_ffff` | 内核态：R/W， 用户态不可访问，下面的都是这样   | MMIO                         |
| `0xffffffc0_0010_0000` ~ `0xffffffc0_7fff_ffff` | 内核态：R/W   | PCIe BAR                     |
| `0xffffffc0_8020_0000` ~ `0xffffffc0_803e_3000` | 内核态：R/X   | kernel `.text`               |
| `0xffffffc0_803e_3000` ~ `0xffffffc0_8277_5000` | 内核态：R     | `.rodata`                    |
| `0xffffffc0_8277_5000` ~ `0xffffffc0_c000_0000` | 内核态：R/W   | data / bss / linear phys map |
